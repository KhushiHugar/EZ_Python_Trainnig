TYPES OF ARGUMENTS
1.POSITIONAL(VARIABLES TAKE EXACT VALUE WRITTEN)
 EXAMPLE:
    def sumofnumbers(a,b):
       print(a)
       print(b)
       return a+b
       ans= sumofnumbers(12,13)
       print(ans)
   
    output: a=12, b=13 25


2.DEFAULT
example: def myself(name,age):
    print(name)
    print(age)
    return None
myself("shree",age=24)

24
